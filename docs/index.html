關於夜雪羽
夜雪羽，起初是個成立於《陰陽師》的休閒寮辦
《決戰！平安京》國際服上線後，同步在平安京裡成立了分寮
2021年4月27日 《陰陽師》的夜雪羽本寮與他寮進行合併之後
《決戰！平安京》的夜雪羽隨即成為了本寮
夜雪羽是個秉持著「休閒、放鬆」精神的寮辦
我們認為遊戲應當為讓人感到放鬆、帶來愉快體驗的事物
而不是束縛時間與自由的枷鎖，這顯然違背了遊戲的本質
因此，夜雪羽希望加入寮辦的寮友們能夠享受遊戲
不僅是在平安京，不管是任何遊戲，都應該盡情歡笑
為自己的人生、帶來多一點的繽紛色彩，以及快樂

社群
為了讓寮友擁有更易於交流的環境
夜雪羽自成立以來即創建專屬 LINE 聊天群組
因應寮友遊戲語音需求，亦創建了專屬的 Discord 頻道
讓寮友在遊戲時有著更方便的語音交流體驗
所有社群仰賴寮友們共同維護、締造優質、良好的交流環境
